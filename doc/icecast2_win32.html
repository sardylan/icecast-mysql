Icecast 2 - Win32 Specific Documentation

The win32 port of icecast2 is a simple command line application, it used to be a UI framework around the core icecast2 server.

Most of the features of icecast2 are available in the win32 port.
A notable absence is IPv6 support.

If there is sufficient user interest we may reintroduce the Icecast GUI and capability to start Icecast as a Windows Service.
